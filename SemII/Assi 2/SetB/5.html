In folder -> routes/web.php Add following route Route::get('/profile/{username?}', function ($username = "Guest") { return "Username: " . $username; }); -------------------------------------------------- How to check output: Run server: php artisan serve Open in browser: 1) With name: http://127.0.0.1:8000/profile/Siddhi Output: Username: Siddhi 2) Without name: http://127.0.0.1:8000/profile Output: Username: Guest
